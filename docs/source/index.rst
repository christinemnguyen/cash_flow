Cash Flow Main
==============

.. automodule:: app
    :members:


Register
========

    Allows user to register for an account after they provide a valid username,
    email, and password. 

Login
=====

    Allows user to login to an account that they created.

Logout
======

    Allows user to logout of an account that they're currently logged into.

Navigation Bar
==============

    Allows user to navigate to other web pages on the website.

Input Credit Card
=================

    Allows user to input credit card information such as their current cash
    back options and the amount they spend per month.

Delete Credit Card
==================

    Allows users to delete credit cards that they no longer need.

Compare Credit Cards
====================

    Allows user to see other credit cards that are confirmed to be good, and
    see if other cards are better or not.

Admin Page
==========

    Allows admin users to delete and create new credit cards that users can
    compare their current credit cards to.
